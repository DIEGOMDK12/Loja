# Sua Loja Digital Pro

Uma loja digital moderna e elegante para vendas via WhatsApp.

## Como Hospedar no GitHub Pages

### Passo 1: Criar Repositorio
1. Acesse [github.com](https://github.com) e faca login
2. Clique em **"New repository"** (novo repositorio)
3. De um nome ao repositorio (ex: `minha-loja`)
4. Deixe como **Public** (publico)
5. Clique em **"Create repository"**

### Passo 2: Enviar o Arquivo
1. No repositorio criado, clique em **"uploading an existing file"**
2. Arraste o arquivo `index.html` para a pagina
3. Clique em **"Commit changes"**

### Passo 3: Ativar GitHub Pages
1. Va em **Settings** (configuracoes) do repositorio
2. No menu lateral, clique em **Pages**
3. Em "Source", selecione **Deploy from a branch**
4. Selecione a branch **main** e a pasta **/ (root)**
5. Clique em **Save**

### Passo 4: Acessar sua Loja
Apos alguns minutos, sua loja estara disponivel em:
```
https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/
```

## Como Usar

### Acessar o Painel Admin
1. Clique no botao de engrenagem no canto inferior direito
2. Digite a senha padrao: `admin123`
3. Altere a senha apos o primeiro acesso!

### Adicionar Produtos
1. Acesse o painel admin
2. Preencha os campos do produto:
   - Nome do produto
   - URL da imagem (use links diretos)
   - Preco original e com desconto
   - Numero do WhatsApp (com DDD e codigo do pais)
   - Descricao do produto
3. Clique em "Salvar Produto"

### Dicas para Imagens
Use servicos gratuitos para hospedar imagens:
- [ImgBB](https://imgbb.com/) - Upload gratuito
- [Imgur](https://imgur.com/) - Upload gratuito
- [Postimages](https://postimages.org/) - Upload gratuito
