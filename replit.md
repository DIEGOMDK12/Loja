# Sua Loja Digital Pro

## Overview
A modern, beautiful digital storefront for selling products via WhatsApp. Features a stunning dark theme with gradients and glass effects, optimized for both mobile and desktop.

## Current Version
**Static HTML Version** - Ready for GitHub Pages hosting

## Features
- Modern gradient and glassmorphism design
- Product catalog with discount badges
- WhatsApp integration for instant sales
- Password-protected admin panel
- CRUD operations for products
- Responsive design (mobile-first)
- Smooth animations and transitions
- No server required - runs entirely in browser

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (vanilla)
- **Fonts**: Google Fonts (Inter)
- **Storage**: Browser LocalStorage
- **Hosting**: GitHub Pages compatible

## How to Use

### For GitHub Pages (Static)
1. Upload `index.html` to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Access your store at `https://username.github.io/repo-name/`

### For Replit (Server Version)
The Flask server runs on port 5000 with SQLite database.
